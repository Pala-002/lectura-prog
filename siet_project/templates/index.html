Sistema Inteligente de Evaluación de Tecnoestrés

Plataforma científica para la evaluación del tecnoestrés mediante un enfoque híbrido que combina RED-TIC, pruebas cognitivas digitales, Learning Analytics y Behavioral Analytics.

Componentes del Sistema

RED-TIC

Escala validada para medir fatiga, ansiedad, escepticismo e ineficacia tecnológica.

Pruebas Cognitivas

Stroop, N-Back, Digit Span, Trail Making y Cognitive Reflection Test.

Learning Analytics

Análisis de tiempo, clicks, cambios de respuesta y patrones de aprendizaje.

Behavioral Analytics

Seguimiento de movimientos, scroll, focus, blur y eventos de interacción.
