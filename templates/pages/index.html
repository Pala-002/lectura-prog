Objetivo del Proyecto

El SIET es una plataforma de investigación diseñada para estudiar la relación entre el uso de tecnologías digitales y el tecnoestrés en población universitaria, evaluando simultáneamente funciones cognitivas afectadas.

Evaluación Cognitiva

Pruebas estandarizadas para medir atención, memoria y función ejecutiva.

Análisis de Datos

Procesamiento estadístico avanzado para identificar patrones y correlaciones.

Confidencialidad

Protección de datos personales bajo estándares éticos de investigación.
